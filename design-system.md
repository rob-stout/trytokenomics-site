## Border Radius

Single style: large rounded (consumer/Apple-adjacent).

```
--radius-sm:  8px    — buttons, badges, small chips
--radius-md:  12px   — nav, install block, FAQ items
--radius-lg:  20px   — cards, widget chrome (matches app 20pt)
--radius-xl:  28px   — hero mock, large showcases
--radius-full: 9999px — pill buttons, progress bars
```

Rule: children always have smaller radius than their parent (nested: 20→12→8).

## Elevation / Depth

Dark mode uses lighter surfaces for elevation. Light mode uses subtle tinted shadows.

```
--shadow-sm:  0 1px 3px rgba(0,0,0,0.12), 0 1px 2px rgba(0,0,0,0.08)
--shadow-md:  0 4px 16px rgba(0,0,0,0.20), 0 2px 6px rgba(0,0,0,0.12)
--shadow-lg:  0 12px 40px rgba(0,0,0,0.30), 0 4px 12px rgba(0,0,0,0.15)
--shadow-glow: 0 0 40px rgba(117,203,245,0.12)   /* dark: blue glow on cards */
```

Light mode: replace rgba(0,0,0,x) with rgba(14,34,51,x * 0.5) for warm-tinted shadows.

## Component Specs

### .btn-secondary

### .card
- Background: `--surface`
- Border: 1px solid `--border`
- Border-radius: `--radius-lg` (20px)
- Padding: `--space-5` (24px)
- Dark mode: `box-shadow: --shadow-glow`
- Light mode: `box-shadow: --shadow-sm`
- Nested inner shadow (light only): `box-shadow: inset 0 1px 0 rgba(255,255,255,0.6), --shadow-sm`

### .bar
- Track: height 6px, border-radius `--radius-full`, background `--track-bg`
- Fill: capsule, height 6px, background `--accent-primary` (or `--accent-deep` for long window)
- CSS transition: `width 0.5s cubic-bezier(0.25,0.46,0.45,0.94)` triggered by `.animated` class
- Pace dot: `::before` pseudo-element, 6px circle, background `--pace-dot`,
  positioned via CSS custom property `--pace-x` (percentage offset), `transform: translateX(-50%)`
  Pace dot only visible when `--pace-x` is between 2% and 98%.

### .ring (SVG)
- Dual concentric SVG rings
- Outer: stroke-width 6, r=36 (in 100x100 viewBox)
- Inner: stroke-width 6, r=22
- Track stroke: `--track-bg`
- Fill stroke: gradient from `--accent-primary` to `--accent-deep` (outer), `--accent-primary` (inner)
- Stroke-linecap: round
- Starts at 12 o'clock (rotate -90deg on the SVG arc)
- Pace dot: `<circle>` element at the angular position of the pace fraction, fill `--pace-dot`, r=3

## Motion System

- Respect `prefers-reduced-motion`: all animations off, transitions collapse to 0ms
- Bar fill on scroll-enter: `width` animates from 0 to final value in 500ms `cubic-bezier(0.25,0.46,0.45,0.94)`
- Pace dot on scroll-enter: after 400ms delay, gentle 2.5s sinusoidal drift ±4px `animation: paceDrift 2.5s ease-in-out infinite`
- Ring fill: stroke-dashoffset animates from 0 to filled in 600ms `cubic-bezier(0.25,0.46,0.45,0.94)`
- Cards scroll-enter: `opacity: 0 → 1`, `translateY(16px → 0)`, 400ms ease-out, 80ms stagger
- FAQ expand: max-height 300ms `cubic-bezier(0.25,0.46,0.45,0.94)` — no height/width animation on other elements
- Button hover: 150ms ease-out `transform` and `filter` only
- Theme toggle: 200ms ease-out on `background-color`, `color`, `border-color`, `box-shadow`
